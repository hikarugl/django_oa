### 1、部门列表接口         
1. 接口地址：/app/dept/
2. 支持格式：json
3. 请求方式：GET
4. 请求参数： 
   | 名称 | 必填 | 类型 | 说明 |
   | ---- | ---- | ---- | ---- |
   | 无   |      |      |      |

5. 返回字段说明

   | 名称  | 类型 | 说明         |
   | ----- | ---- | ------------ |
   | msg   | str  | 响应信息     |
   | code  | int  | 状态码       |
   | count | int  | 部门总数     |
   | data  | list | 部门详细信息 |

6. JSON返回示例

   ```
     成功响应:     
   {
   	'code': 0,
   	'msg': '',
   	'count': 3,
   	'data': [{
   		'department_num': 'ZJB',
   		'department': '总经办',
   		'higher_id': None,
   		'description': 'erwtresgs'
   	}]
   }   
   
   ```
